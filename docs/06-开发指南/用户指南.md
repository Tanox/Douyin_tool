# 用户指南

本文档旨在帮助用户了解如何使用抖音UI定制器，包括安装、配置和功能使用方法。

- [安装指南](#安装指南)

## 安装指南

### 前置要求

- 支持的浏览器：Chrome 80+、Firefox 75+、Edge 80+、Safari 13+
- 安装Tampermonkey或Greasemonkey扩展

### 安装步骤

1. 在浏览器中安装Tampermonkey或Greasemonkey扩展
2. 访问[GitHub发布页面](https://github.com/sutchan/douyin_tool/releases)
3. 下载最新版本的`douyin_ui_customizer.user.js`文件
4. 点击下载的文件进行安装
5. 安装完成后，打开抖音网站即可使用

## 基本配置

安装完成后，用户可以通过点击浏览器扩展图标中的设置按钮进行基本配置：

- 启用/禁用脚本
- 设置默认主题
- 配置UI元素显示
- 设置快捷键

## 功能使用

### 主题切换

点击页面右上角的主题切换按钮，可以在浅色和深色主题之间切换。

### 元素控制

使用元素控制面板可以：
- 显示/隐藏特定UI元素
- 调整元素大小和位置
- 修改元素样式

### 快捷键操作

- `Alt+T`：切换主题
- `Alt+S`：打开设置面板
- `Alt+R`：重置所有设置

## 常见问题

### 脚本无法加载

- 确保已正确安装Tampermonkey或Greasemonkey
- 检查脚本是否启用
- 刷新抖音页面

### 功能不生效

- 确保抖音网站版本与脚本兼容
- 尝试清除浏览器缓存
- 检查是否有其他脚本冲突

# 抖音UI定制器 - 项目简介

## 1. 项目概述

抖音UI定制器是一个功能强大的浏览器用户脚本(UserScript)，专为定制抖音网页界面而设计。该脚本允许用户自定义抖音网页版的外观和功能，提供个性化的浏览体验。

### 1.1 项目定位
- **面向用户**: 希望自定义抖音界面的普通用户和高级用户
- **核心价值**: 提供灵活的UI定制能力，增强用户体验
- **技术属性**: 浏览器用户脚本，基于JavaScript和CSS开发

### 1.2 发展历程
- **2024.03**: 项目启动，完成基础架构搭建
- **2024.04**: 发布首个版本，支持基本主题切换
- **2024.05**: 添加元素控制和布局调整功能
- **2024.06**: 优化性能和用户体验，完善文档

## 2. 核心功能

抖音UI定制器提供了丰富的功能模块，用户可以根据自己的需求进行个性化定制：

### 2.1 主题切换
- 支持浅色/深色主题切换
- 自定义主题颜色方案
- 预设多种主题风格

### 2.2 元素控制
- 显示/隐藏界面元素
- 调整元素大小和位置
- 自定义元素样式

### 2.3 布局调整
- 自定义页面布局结构
- 调整内容区域大小
- 优化视频播放体验

### 2.4 动画效果
- 添加/移除界面动画
- 调整动画速度和效果
- 自定义过渡效果

### 2.5 用户配置
- 保存个性化配置
- 导出/导入配置文件
- 配置预设管理

### 2.6 数据同步
- 跨设备配置同步
- 配置版本管理
- 自动备份配置

## 3. 技术架构

### 3.1 技术栈
- **开发语言**: JavaScript (ES6+)
- **样式技术**: CSS, CSS变量
- **构建工具**: Vite
- **存储方案**: localStorage, GM_setValue/GM_getValue

### 3.2 系统架构
```
┌─────────────────────────┐
│      用户界面层         │
│  (UI Manager, Components)│
├─────────────────────────┤
│      功能模块层         │
│ (Theme, Elements, Layout)│
├─────────────────────────┤
│      核心服务层         │
│ (Config, Storage, Event) │
├─────────────────────────┤
│      基础工具层         │
│ (Utils, Helpers)         │
└─────────────────────────┘
```

### 3.3 关键技术点
- **模块化设计**: 功能模块解耦，便于扩展和维护
- **事件驱动**: 基于发布-订阅模式的事件系统
- **性能优化**: 防抖、节流、虚拟DOM等技术
- **兼容性处理**: 支持主流浏览器和不同版本

## 4. 安装与使用

### 4.1 安装条件
- 现代浏览器(Chrome, Firefox, Edge等)
- 安装用户脚本管理器(Tampermonkey, Violentmonkey等)

### 4.2 安装步骤
1. 安装用户脚本管理器扩展
2. 访问项目GitHub页面
3. 点击"安装脚本"按钮
4. 确认安装并启用

### 4.3 基本使用
1. 打开抖音网页版
2. 点击脚本图标打开控制面板
3. 选择需要的功能模块
4. 调整参数并应用

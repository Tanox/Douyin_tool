## 架构概述

抖音Web端界面UI定制工具采用模块化的架构设计，主要由以下几个核心模块组成，各模块之间通过清晰的接口进行通信，保证了代码的可维护性和可扩展性。

## 核心模块

### 1. 主模块 (main.js)

作为整个工具的入口点，负责初始化各个子模块，协调它们之间的工作。

### 2. 配置管理模块 (config.js)

负责管理用户配置和默认设置，提供配置的读写功能。

### 3. UI管理模块 (ui_manager.js)

核心功能模块，负责处理界面元素的操作，包括：
- 主题切换
- 元素隐藏/显示
- 布局调整

### 4. 工具模块 (utils/)

提供通用功能和辅助方法：
- dom.js - DOM操作相关工具函数
- storage.js - 存储相关工具函数

### 5. 样式模块 (styles/)

包含不同主题的样式定义：
- default.css - 默认主题样式
- dark.css - 暗黑主题样式
